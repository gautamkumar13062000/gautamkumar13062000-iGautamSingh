  1. HTML Boilerplate: This is the basic html boilerplate code 
    
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    
</body>
</html>
  2. Headings: There are six headings available in HTML, H1 is the largest among all, and H6 is the smallest. 
    <h1>Heading 1</h1>
  3. div tag:div tag or division tag is used to make blocks or divisions in the document.
    4. <div> This is div block </div>
  4. span tag: span is a container for inline content 
    <span> This is span block </span>
  5. p tag: Paragraph 
     <p>This is a paragraph </p>
  6. <img> tag: It is used to embed or import image in a webpage. 
    <img src="Source_of_image" alt="Alternate text">
  7. code tag: code tag is used to represent source codes 
    
<code>
    import python
</code>
  8. <b> tag: 
    <b>I'm bold text</b>
  9. <audio> tag: It is used to embed sound content in the document. 
    
<audio controls>
<source src="demo.mp3" type="audio/mpeg">
Your browser does not support the audio element.
</audio>
  10. <video> tag: It is used to embed video in the webpage. 
    
<video width="480" height="320" controls>
<source src="demo_move.mp4" type="video/mp4">
Your browser does not support the video tag.
</video>
